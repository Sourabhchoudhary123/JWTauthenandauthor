# JWT Authentication API

This project demonstrates how to implement **JWT (JSON WEB TOKEN) authentication** in a Node.js + Express application,
It includes routes and user login, token generation, and accessing protected

--

##Features

-- User registration & login
-- JWT token generation
-- Protected routes with middleware
-- Token verification & decoded payload response
